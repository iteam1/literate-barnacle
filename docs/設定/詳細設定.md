# 詳細設定

## 概要

このページでは動作を細かく調整するための上級者向け設定項目について説明します。

## タイムアウト設定

処理のタイムアウト時間を秒単位で指定します。

- 設定キー: `timeout`
- デフォルト値: `30`
- 設定例: `timeout: 60`

## 並列処理数

同時に実行する処理の数を指定します。値を大きくすると速くなりますが、メモリ使用量も増加します。

- 設定キー: `workers`
- デフォルト値: `2`
- 推奨値: CPUコア数以下
- 設定例: `workers: 4`

## キャッシュ設定

キャッシュの有効・無効と保存期間を設定します。

- 有効・無効の設定キー: `cache_enabled`（デフォルト: `true`）
- 保存期間の設定キー: `cache_ttl`（単位: 秒、デフォルト: `3600`）
- 設定例:

```
cache_enabled: true
cache_ttl: 3600
```

設定を変更した後は、アプリケーションを再起動してください。
