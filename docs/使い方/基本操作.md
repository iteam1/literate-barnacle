# 基本操作

## 概要

このページでは起動から終了までの基本的な操作手順を説明します。

## 目次

1. [起動方法](#起動方法)
2. [メインメニューの操作](#メインメニューの操作)
3. [終了方法](#終了方法)

## 起動方法

以下の手順でアプリケーションを起動してください。

1. ターミナルを開く
2. プロジェクトディレクトリへ移動する
3. 起動コマンドを実行する

## メインメニューの操作

起動後、メインメニューが表示されます。

1. 目的の操作をメニューから選択する
2. 画面の指示に従って入力を行う
3. 処理が完了するまで待つ
4. 処理結果を確認する

## 終了方法

操作完了後は、以下の手順で正しく終了してください。

1. メインメニューに戻る
2. 終了を選択する
3. 確認メッセージに応答する
