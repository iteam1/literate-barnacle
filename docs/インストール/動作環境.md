# 動作環境

## 概要

このページでは動作確認済みの環境と必要なソフトウェアについて説明します。

## 目次

1. [対応OS](#対応os)
2. [必要なソフトウェア](#必要なソフトウェア)
3. [推奨スペック](#推奨スペック)

## 対応OS

以下のOSで動作を確認しています。

- Windows 10 / 11
- macOS 12 以上
- Ubuntu 22.04 以上

## 必要なソフトウェア

- Python 3.9 以上
- pip 最新バージョン
- インターネット接続環境

## 推奨スペック

- CPU: 2コア以上
- メモリ: 4GB 以上
- ディスク空き容量: 1GB 以上
