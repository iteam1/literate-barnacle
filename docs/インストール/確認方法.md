# 確認方法

このページではインストール後の動作確認手順を説明します。

## バージョン確認

インストールが正常に完了したか確認するには以下の手順を実施してください。

1. ターミナルを開く
2. バージョン確認コマンドを実行する
3. バージョン番号が表示されれば成功です

## 動作テスト

バージョン確認後、簡単な動作テストを行ってください。

1. サンプルコマンドを実行する
2. 期待通りの出力が得られるか確認する
3. ログファイルにエラーがないか確認する
